## 📂 Featured Projects

### 🏠 [Austin Animal Center Adoption Modeling](https://github.com/lindseyschae/adopt)
Predicting animal shelter outcomes using **170,000+** intake records from Austin Animal Center.
- Engineered time-based and categorical features from intake/outcome records
- Built classification models achieving **F1 scores up to 0.80**
- Identified key drivers of longer shelter stays to support adoption strategy decisions
- Delivered findings through structured visualizations and reporting

**Stack:** Python · Pandas · Scikit-Learn · XGBoost · CatBoost · LightGBM · Matplotlib · Seaborn

---

### 🐕 [CBARQ Dog Breed Behavior Analysis](https://github.com/lindseyschae/Puppers)
Analyzing behavioral survey data to evaluate whether breed characteristics predict herding behavior.
- Performed EDA and feature encoding on categorical breed data
- Built supervised ML models to predict behavioral scores
- Visualized trends and interpreted model outputs to communicate statistical findings clearly

**Stack:** Python · Pandas · Scikit-Learn · Seaborn · Plotly · Regression Models
